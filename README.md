# Lightboard

Shopify storefront built with [Hydrogen](https://hydrogen.io) and deployed on [Oxygen](https://shopify.dev/docs/custom-storefronts/oxygen).

## Stack

- [Hydrogen](https://hydrogen.io) — Shopify's React-based storefront framework
- [Oxygen](https://shopify.dev/docs/custom-storefronts/oxygen) — Shopify's edge hosting platform
- [Vite](https://vitejs.dev) — build tool
- [React Router v7](https://reactrouter.com) — routing

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Every push to `main` automatically deploys to Oxygen via GitHub Actions.
